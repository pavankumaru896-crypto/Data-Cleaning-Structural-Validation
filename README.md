# Data Cleaning & Structural Validation

## Objective

Transform raw, unstructured data into a reliable format for analysis by identifying and correcting duplicate entries, missing values, and inconsistent formats while preserving data integrity.

## Dataset Issues

* Duplicate records
* Missing values
* Invalid entries
* Inconsistent city names
* Mixed date formats

## Cleaning Process

1. Removed duplicate records.
2. Filled missing email addresses.
3. Filled missing phone numbers.
4. Replaced missing age values with median values.
5. Corrected invalid age values.
6. Standardized city names.
7. Converted date formats to YYYY-MM-DD.
8. Validated and cleaned revenue data.

## Tools Used

* Microsoft Excel

## Results

* Improved data quality and consistency.
* Standardized formats across all records.
* Generated a clean dataset ready for analysis.
